##### 상세

`Next.JS` 의 기본 **API**와 **Github**에서 제공하는 [**users API**](https://api.github.com/users)를 활용해 **Github**의 유저 페이지를 바탕으로 **개인의 개발에 대한 경험을 요약하는 페이지**를 만들어보는 **토이 프로젝트**.

##### 기술

- `React.js`
- `Next.js`
